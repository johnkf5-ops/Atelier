# project-proposal-real-examples

WHEN TO USE: Loaded into the Package Drafter alongside `project-proposal-structure.md`. Where the structure file gives the generic six-beat shape and two worked examples (Creative Capital, Guggenheim), this file gives type-specific templates, verbatim recipient examples, and anti-examples drawn from real funder pages. Use this file to pick the *right* template before drafting; use the structure file to fill it.

---

## What a project proposal actually is

A project proposal is a short prose document that tells a panel what you intend to do with their money, time, or wall, and why you specifically should be the one doing it. It is not an artist statement. The artist statement explains the practice in general; the proposal commits to a specific, time-bounded, deliverable-bearing action. The most important rule is that different opportunity types want structurally different proposals — a state arts council fellowship asks for a project plan with a timeline and deliverables, a photography competition asks for curatorial framing of finished work, a residency asks what would be made *during* the residency in *that* place, a book grant asks for a book plan, a foundation grant asks how unrestricted funds will sustain the practice, and a public-art RFQ asks for evidence of fit and capability. Treating every prompt as a generic "submission package" produces text that loses against the rubric the panel is actually scoring against. The proposal must answer the prompt the funder wrote, in the structure that funder reads against, at the length that funder allows.

## Six proposal type templates

### 1. State arts council fellowship (NEA / NYSCA / state councils)

**Ground truth:** [NEA Grants for Arts Projects: Applicant Resources](https://www.arts.gov/grants/grants-for-arts-projects/applicant-resources), [NYSCA Support for Artists FY2025 manual](https://nysca.org/downloads/files/FY2025_NYSCA_SupportArtists_AppManual.pdf), [NYSCA/NYFA Artist Fellowship Photography](https://www.nyfa.org/awards-grants/nysca-nyfa-artist-fellowship/).

**What it funds:** New work by a named artist, in a defined period, often through a fiscal sponsor. NYSCA Support for Artists awards are $10,000 to a 501(c)(3) fiscal sponsor on the artist's behalf; NYSCA/NYFA Artist Fellowships are unrestricted $8,000 awards.

**Required sections:**
- **Project description** (NEA Project Description: 5,000 characters max; NYSCA narrative comparable). What the project is, what activities will take place during the period of performance, why this work, why now.
- **Engagement / public-benefit statement** (NEA: separate 2,000-character field). For state councils, the public-benefit angle is load-bearing — the funder is a public-money agency.
- **Timeline.** Months, not "phases."
- **Budget.** Itemized for NEA; for state councils, a budget summary tied to deliverables.
- **Work samples + sample descriptions.**
- **CV / resume.**
- **Bio / artist statement** (separate field — do not duplicate inside the proposal).

**Length:** NEA project description allows ~750 words. NYSCA narrative similar. Write to the cap, not under it; panels read these as applicant capability tests.

**Strong-proposal moves:**
- Open with one sentence that names the project, the medium, and the geography.
- Address each NEA or council review criterion explicitly inside the narrative (do not assume the panel will infer).
- Specify what is NEW about this project relative to prior work — state councils fund forward, not backward.
- Name the fiscal sponsor (for NYSCA) and the relationship in one clause.

**Common rejections:** Project reads as continuation of practice, not a new project. No specific deliverables. Public-benefit field treated as boilerplate. Budget line items not mapped to deliverables. Duplication of artist-statement content inside the project description.

### 2. Photography competition portfolio submission (ILPOTY, Critical Mass, Aperture Portfolio Prize, IPA)

**Ground truth:** [ILPOTY 2025 Rules](https://www.internationallandscapephotographer.com/index.php/awards/rulesall), [Critical Mass entry details](https://www.photolucida.org/critical-mass/entry-details/), [Aperture Portfolio Prize](https://aperture.org/calls-for-entry/portfolio-prize/).

**What it awards:** Recognition, publication, exhibition, sometimes cash. Aperture Portfolio Prize: $5,000 first prize, four $1,000 shortlist prizes, group show at AIPAD. Critical Mass: Top 50 selection; jurors include curators who acquire and editors who publish.

**Required sections:**
- **Portfolio of finished images.** ILPOTY: min 4 entries for Photographer of the Year eligibility. Critical Mass: min 7, typically 10 from one body of work. Aperture: 10–15 images.
- **Project / artist statement** (where required). Critical Mass: 250 words about the specific body of work. Aperture: 1,500 characters (~250 words). ILPOTY: no written statement — pure visual jurying.
- **Bio.** Aperture: 1,500 characters. **CV.** Aperture: 2,000 characters or single-page PDF.

**Length:** 250 words is the dominant ceiling. Every sentence load-bearing.

**Strong-proposal moves:**
- Curatorial framing of an *existing, completed* body of work, not a project plan.
- Name the body of work as a titled series.
- One sentence on subject, one on approach/method when load-bearing, one on what unifies the 10 images as a series, one on stakes.
- No timeline, no budget, no deliverables.

**Common rejections:** Statement reads as generic artist statement rather than frame for these specific 10 images. Statement explains what each image is rather than what the series argues. Boilerplate lineage paragraph. Mismatch between statement claim and image set.

### 3. Residency application (MacDowell, Yaddo, Headlands, Vermont Studio Center, Light Work)

**Ground truth:** [MacDowell Application Guidelines](https://www.macdowell.org/apply/application-guidelines), [MacDowell Work Sample Requirements](https://www.macdowell.org/apply/work-sample-requirements), [Headlands AIR program](https://www.headlands.org/program/air/), [Vermont Studio Center Apply](https://vermontstudiocenter.org/apply), [Light Work AIR](https://www.lightwork.org/air/).

**What it funds:** Time, space, sometimes a stipend, in a specific place. MacDowell: up to 8 weeks, studio + meals + travel reimbursement. Headlands: 4–10 weeks, fully sponsored, $200/week stipend, chef-prepared meals. Light Work: one month, $5,000 stipend, apartment, private digital studio and darkroom, plus a feature in *Contact Sheet*.

**Required sections:**
- **Project description.** MacDowell does not publish a word count for the visual-arts project description prompt — applicants typically write 250–500 words. The prompt asks what the applicant hopes to accomplish during the residency.
- **Work samples.** MacDowell: 8 examples completed within last 4 years (preferred file size <3MB each). Headlands: up to 12 work samples. Vermont Studio Center: 5 images for visual artists, no age restriction.
- **CV.**
- **References** (some residencies, including MacDowell, require recommendation letters).

**Length:** Project description is short — most residencies. The work samples carry most of the weight. MacDowell explicitly states the sole criterion is artistic excellence and that the panel scores blind from work samples first.

**Strong-proposal moves:**
- Specify what would be *done* during the residency weeks, not what the larger project is. The grant is for the residency window, not for the project lifetime.
- Connect the proposal to the place when load-bearing (Headlands evaluates "potential to develop creative practice and impact the community at Headlands and beyond" — place-specificity matters there). At MacDowell, place-specificity matters less; the panel weights the work itself.
- For darkroom-equipped residencies (MacDowell's analog black-and-white darkroom; Light Work's private darkroom and digital studio), name the facility and what would be done in it. This signals fit.
- Brief context paragraph on the larger project the residency advances — but the bulk of the description is what gets done in those weeks.

**Common rejections:** Description of a project so large the residency window cannot meaningfully advance it. No relationship to the residency facility. Project that could happen anywhere — residencies fund the *being there*. Overlong context that crowds out what would actually happen during the residency.

**Verbatim recipient project descriptions (MacDowell):**

> Tanya Marcuse, 2018: completed "a mock-up of her forthcoming book comprised of three major projects, *Fruitless/Fallen/Woven*, with Radius Press"; created "a series of formal portraits of fellows and staff in the MacDowell community"; installed a 66 x 126" studio print from the *Woven* series at the James Baldwin Library. ([MacDowell — Tanya Marcuse](https://www.macdowell.org/artists/tanya-marcuse))

> Kristen Joy Emack: "photographed vintage family photos in the natural landscape on the MacDowell property surrounding her studio. These images will accompany the creative nonfiction story she wrote there." Used her residency time to "sequence her book dummy for her long-term documentary series, 'Cousins.'" ([MacDowell — Kristen Joy Emack](https://www.macdowell.org/artists/kristen-joy-emack))

These published descriptions are short, name a deliverable (book mockup, sequenced book dummy), name a method (vintage family photos in the natural landscape, formal portraits of fellows and staff), and connect to the residency context where load-bearing (the MacDowell property, the James Baldwin Library, the fellow community).

### 4. Photo book grant (Aperture Portfolio Prize / PhotoBook Awards, Lucie Foundation Photo Book Prize, Anamorphosis Prize)

**Ground truth:** [Paris Photo–Aperture PhotoBook Awards](https://aperture.org/calls-for-entry/photobook-awards/), [Lucie Foundation Photo Book Prize](https://www.luciefoundation.org/lucie-photo-book-prize/), [Anamorphosis Prize](https://anamorphosisprize.com/).

**What it funds / awards:** Aperture First PhotoBook: $10,000 to the photographer. Anamorphosis: $10,000 to a self-published book in an edition of 50+. Lucie: cash prizes; submissions accept dummies and finished books.

**Required sections:**
- **The book itself, or a book dummy.** This is the primary submission. For Anamorphosis, the finished self-published book must be physically submitted. For Lucie, a PDF cover-to-cover (max 60MB) of the complete book in layout form.
- **Project / book statement.** Short. Aperture-comparable awards: 1,500 characters. The statement frames the book — concept, scope, sequence logic, scale, edition.
- **Selection of images.**
- **CV / bio.**

**Length:** Statements are short; the book is the proposal. The statement is a frame for jurors who will then read the book.

**Strong-proposal moves:**
- Name the working title.
- One sentence on subject and concept.
- One sentence on the book object decisions: page count, trim size, paper, binding, cover, edition size — these are book-specific decisions that signal the photographer is making a book, not just submitting a portfolio in book form.
- One sentence on sequencing logic — what argument the order of images makes.
- One sentence on publisher relationship if any (verbal agreement, named publisher, self-published).
- For book grants the deliverable is a *physical book*. Make the physical book real in the statement.

**Common rejections:** Statement frames the work as a portfolio rather than as a book object. No mention of sequence, scale, page count, edition. Confuses a body of work with a book — they are not the same thing. No publisher path.

**Verbatim recipient framings:**

> *A Study on Waitressing* by Eleonora Agostini (Aperture First PhotoBook 2025): "a conceptual study explor[ing] performative labor and self-presentation through fictionalized images of a waitress (the artist's mother). Divided into six chapters with staccato sequences, the work layers mundane gestures with semitransparent pages containing handwritten Italian notes." ([Aperture 2025 PhotoBook Awards](https://aperture.org/editorial/announcing-the-winners-of-the-2025-photobook-awards/))

> *Wild Pigeon* by Carolyn Drake (Anamorphosis 2015): "shows the people and landscapes of the remote, rapidly changing Xinjiang Uyghur Autonomous Region of China. Alongside the images, shot between 2007 and 2013, are also collages and drawings made by local people, and references to the story Wild Pigeon by Uyghur author Nurmuhemmet Yasin." ([Anamorphosis 2015](https://anamorphosisprize.com/winner-2015/))

Structural moves to borrow: time period of shooting named, geography named, *book object* characteristics named (six chapters, semitransparent pages, collages), the argument the book makes named.

### 5. Foundation grant (Pollock-Krasner, Aaron Siskind / VMFA, Light Work, En Foco)

**Ground truth:** [Pollock-Krasner: How to Apply](https://www.pkf.org/apply/how-to-apply/), [VMFA Aaron Siskind Award](https://www.vmfa.museum/vmfa-aaron-siskind-award-for-photography), [En Foco Artist Fellowship](https://enfoco.org/artist-fellowship/).

**What it funds:** The artist's continued practice. Pollock-Krasner grants are unrestricted, up to $50,000, usable for "create new work, acquire supplies, rent studio space, prepare for exhibitions, attend a residency, and offset living expenses." The grant supports the practice; it does not require a discrete project.

**Required sections by funder:**
- **Pollock-Krasner.** Cover letter: "A cover letter stating for what specific purpose (professional, personal, and/or medical) you require funds and in what amount." This is the load-bearing document — no separate project description. Plus: one-page statement describing the 10 submitted images, CV with exhibition record, 10 images + ID list.
- **Aaron Siskind / VMFA.** 10 images of recent work, artist statement up to 250 words, CV up to 500 words.
- **En Foco.** Written description of practice up to 300 words (current cycle), addressing subject matter, context, content, location, personal relevance.

**Length:** Short. CV and image set carry weight; the cover letter / statement is a frame.

**Strong-proposal moves:**
- Pollock-Krasner: state the amount requested and *specific purposes* (e.g., "$25,000: $12,000 toward studio rent for 12 months, $8,000 toward print production for confirmed exhibition at [venue], $5,000 toward unreimbursed medical"). This is what the foundation explicitly asks for.
- Siskind / Light Work / En Foco: the statement frames current work, names the body of work submitted, locates it in subject and approach. Not a future project.
- Personal relevance is explicit at En Foco; name it.

**Common rejections:** Pollock-Krasner: no specific dollar amount; vague "to support my practice." Siskind: 250 words used as generic artist statement instead of frame for the 10 submitted images. En Foco: no personal-relevance throughline.

### 6. Public art commission / RFQ (city/county arts commissions)

**Ground truth:** [Salt Lake Public Art: Applying to Calls](https://saltlakepublicart.org/wp-content/uploads/2016/02/Applying-to-Calls-for-Artists.pdf), [Arts Huntsville Public Art Proposals Workbook](https://www.artshuntsville.org/wp-content/uploads/2023/01/Public-Art-Proposal-Workbook.pdf).

**What it funds:** Selection into a pre-qualified pool, or directly into a commission. RFQs are usually two-stage: qualifications submission first (this template), then a paid concept-design phase for shortlisted artists.

**Required sections:**
- **Letter of interest** (one page). Why this project, how the practice connects to the site/brief. NOT a design proposal — design comes in stage two.
- **Resume / CV.**
- **Up to 10 images** of previous work with annotations: title, date, location, materials, art budget, brief description (~5 sentences each).
- **References.**

**Selection criteria (typical weighting):** Artistic/technical quality of past work ~50%, professional experience ~30%, relevance to commissioning organization ~20%.

**Strong-proposal moves:**
- Letter of interest is *site-responsive* without being a design proposal — read the brief, reference its specifics, name what about the practice fits this specific commission.
- Image annotations include *budgets* of past work; panels evaluate fiscal capability as much as aesthetic.
- Photography RFQs: emphasize installation and presentation experience, not just image-making.

**Common rejections:** Letter jumps to design ideas (stage one is qualifications, not concept). Image annotations missing budgets. No site-responsiveness. Past-work images are gallery prints rather than installed public works.

## ANTI-EXAMPLES — what NOT to write

### Anti-example 1: submission letter masquerading as project proposal

> "I am pleased to submit my portfolio for consideration for the [State Council] Fellowship. Enclosed please find ten images representing my recent landscape photography, my CV, my artist statement, and a list of exhibitions. The work submitted reflects my twenty-year practice in the Zone System tradition. I look forward to the panel's consideration."

**Why it fails:** A state arts council fellowship funds NEW work. This text describes nothing the applicant intends to do. It is a cover note, not a proposal. There is no project, no timeline, no deliverable, no budget. The panel reads this and has nothing to score against the funder's criteria.

### Anti-example 2: lineage paragraph dropped into a proposal

> "The proposed work sits in the lineage of Ansel Adams and the Zone System tradition, extended through the cibachrome practice of Galen Rowell and the large-format work of Clyde Butcher. It enters the conversation around landscape-as-witness exemplified by QT Luong's national-park survey and Michael Frye's Yosemite work."

**Why it fails:** Lineage belongs in the artist statement. In a proposal, lineage paragraphs displace the load-bearing content (what is the project, what gets made, when, with what funds). A panel reading a 250-word Critical Mass statement that opens with two sentences of lineage has lost 50 words of project description. Reserve lineage for the statement; use the proposal to commit to specifics.

### Anti-example 3: method / gear boilerplate copied from artist statement

> "I shoot primarily on large-format film, exposing 4x5 transparencies that are drum-scanned and printed as archival pigment prints. My practice draws on the Zone System for previsualization. I use a Linhof Master Technika and Schneider lenses."

**Why it fails:** Technique paragraphs only belong in a proposal when the technique *justifies the project*. "I will return to three Maine rivers across freeze-thaw cycles in 4x5 color negative because the format's tonal range is required to register the ice-water-bare-rock contrast at the project's exhibition scale" is method in service of project. A general gear paragraph is not.

### Anti-example 4: third-person curator-essay register

> "Knopf's photographs survey the American West with quiet authority, attending to the long traditions of landscape photography while pushing into territory the lineage has not yet mapped. His commercial-gallery register meets the durational-return structure of Sugimoto in a body of work whose emotional weight emerges from its restraint."

**Why it fails:** This is the register of an essay *about* an artist, not a proposal *by* an artist. Most proposal forms (NEA, NYSCA, Pollock-Krasner cover letter, MacDowell project description, residency proposals) are first-person documents from the applicant. The third-person curator voice signals that the applicant cannot speak about their own work plainly. Some competition statements (Critical Mass, Aperture) accept first- or third-person, but in those cases first-person is more direct and reads less ghost-written.

### Anti-example 5: em-dash overuse and rhetorical flourish

> "The project, a decadal return to three rivers, proposes to do what no prior body of work has attempted at this scale: to render climate-change-as-witness in the photographic register that the medium has, since Ansel Adams, been preparing itself to deliver."

(The original draft of this sentence used three em-dashes.) Em-dashes as a structural crutch read as AI-generated. Rhetorical flourish ("the medium has been preparing itself") reads as bluster. Panels read 30+ proposals in a sitting; flat declarative prose with concrete nouns wins.

## Voice rules distilled (proposal-specific)

1. First-person for grants, fellowships, residencies, foundation cover letters. Either person for competition statements; first-person reads more direct.
2. Zero em-dashes. Use commas, periods, or parentheses. Em-dash overuse is the strongest tell of generic AI prose; panels notice.
3. Section headers acceptable when the form allows them (NEA narrative, NYSCA narrative, longer foundation cover letters). Prose-only when the form is a single text box (Critical Mass, Aperture, En Foco).
4. Specificity over generality. Real project, real timeline in months, real deliverables with counts, real venues by name when known.
5. No lineage paragraph in a proposal. Lineage lives in the artist statement.
6. No method or gear paragraph repeated from the artist statement. Method appears only when it justifies the project — when a particular format, process, or facility is load-bearing for what is proposed.
7. Deliverables must be countable. Number of works, page count for monograph, edition size, exhibition scale in running feet, residency outputs in defined units.
8. "Why now" answered with a specific reason — a closing window, a confirmed venue, a body of work at the point of needing publication, a site that will not be accessible later. Not a generic urgency claim.
9. "Why this funder" addressed in at least one clause. Why MacDowell rather than Yaddo. Why NYSCA rather than NEA. Why Aaron Siskind rather than Pollock-Krasner. Panels notice when the application reads addressed-to-them.
10. Match the form's word/character cap. Write to the cap. Do not run under by 40%; do not run over.
11. Banned phrases (carried from the artist-statement rules): "sits in the lineage of," "commercial-gallery register," "the medium has been preparing itself," "quiet authority," "emotional weight."
12. The panel reads the proposal, then the work samples. The proposal must make a claim the work samples deliver on. Mismatch is the single most common rejection cause.

## Type-routing logic (for the drafter's classifier)

Classify the opportunity by signals in the funder name, opportunity name, and entry mechanism. Apply in order; first match wins.

| Signal in opportunity | Template |
|---|---|
| Funder is a state arts council (NYSCA, Nevada Arts Council, etc.) OR funder is NEA OR opportunity name contains "Artist Fellowship" with a state council as funder | **State arts council fellowship** |
| Entry mechanism is a portfolio of images via a competition platform (Picter, CallForEntry, Submittable competition listing) AND opportunity name contains "Photographer of the Year" / "Awards" / "Portfolio Prize" / "Critical Mass" / "IPA" / "FAPA" / "ILPOTY" / "OPOTY" / "NLPA" | **Photography competition portfolio** |
| Opportunity is a named residency (MacDowell, Yaddo, Headlands, Vermont Studio Center, Light Work, Banff, Djerassi) OR opportunity name contains "Artist in Residence" / "Residency" | **Residency application** |
| Opportunity name contains "Book Prize" / "Book Award" / "PhotoBook" / "Monograph" OR the deliverable is a published book | **Photo book grant** |
| Funder is a named foundation (Pollock-Krasner, Aaron Siskind / VMFA, Howard Foundation, En Foco, Light Work AIR-stipend component) AND grant is unrestricted-practice rather than project-specific | **Foundation grant** |
| Opportunity is a "Public Art Call" / "Call for Artists" / "RFQ" / "RFP" issued by a city, county, transit authority, or arts commission | **Commission / public art RFQ** |
| Funder is Creative Capital, Guggenheim, USA Fellowship, Joan Mitchell, MacArthur — bespoke majors | **Use `project-proposal-structure.md` directly** (these have published handbooks the drafter should follow verbatim) |

Edge cases:
- A residency that *also* has a publication output (Light Work + *Contact Sheet*) is still primarily a residency proposal; mention the publication as a downstream deliverable, do not let it dominate.
- A foundation grant that is project-restricted (rare but exists, e.g., specific Light Work cycles) routes to **state arts council fellowship** structurally — both are project-restricted with deliverables.
- A photography competition that *also* requests a project statement (Critical Mass, Aperture) is still a **competition** — the statement is curatorial framing of finished work, not a project plan.
